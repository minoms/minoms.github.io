Searching

To open searches, click on the magnifying glass in the upper right corner of the main screen. Click it again to hide the search options (which won’t reset your search, if there is a search still active the magnifier icon will appear filled in). The first row of the basic search has:

• A plus button to add more search rows

• A favorites filter – shows only favorites or all

• A ratings filter – shows recipes greater than or equal to the number of stars selected.

• A reset search button which sets all the search options back to nothing

The first row of the search has:

• Button to add more search rows

• Favorites filter – shows only favorites or all

• Ratings filter – shows recipes greater than or equal to the number of stars selected

• Show hidden recipes – default is to hide hidden recipes, you can also show all, or show just hidden recipes

• A reset button which resets all search options and shows all recipes

Search Line:

• Delete button – removes that search row (not on the first row)

• Text field for the search value

• Clear button which clears that row’s text field

• And/Or selection – indicates how this search line connects to the previous line

• The search type selection

Search Type Selection

• Description – searches the short note field of the main recipe screen

• Ingredient – searches all ingredient fields for the entered word(s). When searching by ingredient the matching ingredients are displayed below the title, that way you can see from that main screen exactly what matched. In the example here, you can see if you search chicken, the search picks up every ingredient with the word chicken, including chicken broth.

• Tags – tags are not pulled from online recipes and need to be selected on the fourth recipe tab. You can add your own custom tags as well as delete the default tags.

• Title (default)

Sorting

MiNoms remembers the way you sorted last, even when you exit the app, that way you don’t always have to reorder the recipes the way you like every time you open the app. To show the sort menu click the arrows in the upper right corner, and click the same button to hide it. The sort options are:

• Favorites – all your favorite recipes at the top

• Frequently Used

• Newest – the latest additions

• Recently Viewed – ordered by the last time you entered the recipe screen

• Star Rating – favorites (or least) first

• Title (default)
